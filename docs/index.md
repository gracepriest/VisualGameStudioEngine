## Components

### Game Engine
A Raylib-based 2D game engine with:
- Entity Component System (ECS)
- Scene management
- Prefab system
- Physics queries
- Debug overlay

### BasicLang Compiler
A custom programming language with:
- VB-like syntax
- Full OOP support
- LINQ queries
- Async/Await
- LSP integration

### Visual Game Studio IDE
A full-featured development environment with:
- IntelliSense and code completion
- Debugging with breakpoints
- Code folding and navigation
- Refactoring tools
